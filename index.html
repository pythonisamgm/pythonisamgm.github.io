Portfolio

Con una sólida base en Java, desarrollo de APIs RESTful y un dominio avanzado en Spring Web, manejo con destreza la construcción de aplicaciones seguras y escalables. Mi experiencia con MySQL y SQLite me permite diseñar y gestionar bases de datos relacionales efectivas y eficientes, mientras que el uso de Junit5 y Mockito garantiza pruebas robustas y fiables para asegurar la calidad del software. Tengo experiencia con herramientas de desarrollo y colaboración como Postman para pruebas de API, XAMPP para el entorno de desarrollo local, y Git y GitHub para la gestión del código fuente y colaboración en proyectos. Trello y Miro son parte integral de mi enfoque en metodologías ágiles, optimizando la gestión y visualización de proyectos, la colaboración en equipo y la planificación estratégica.. Mis habilidades se extienden al desarrollo backend con Flask y FastAPI en Python, y al manejo de SQLAlchemy para la gestión de bases de datos en el ecosistema Python. Selenium para web scraping y Uvicorn como servidor ASGI completan mi capacidad para manejar tareas de automatización y despliegue. Docker me permite crear y gestionar contenedores, asegurando una integración continua y despliegues consistentes. Este amplio stack tecnológico no solo demuestra mi habilidad para adaptarme a diferentes entornos de desarrollo, sino también mi compromiso con la excelencia y la innovación en la creación de soluciones tecnológicas efectivas y eficientes..
